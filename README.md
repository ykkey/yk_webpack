# wk_webpack
====

個人的webpack設定

## 機能
- js babelで書きやすく
- css sassで管理
  autoprefixerで自動ベンダープレフィックス追加
  MiniCssExtractPluginでcss書き出し
- pug meta情報は別で管理
- ローカルサーバー自動立ち上げ
- buildで書き出し


## DEMO
```
npm install
npm run start // develop
npm run build // production
```
